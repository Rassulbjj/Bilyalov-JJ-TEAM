# BILYALOV JJ TEAM — Landing Page

Premium one-page landing site for a Brazilian Jiu-Jitsu academy. Black, white, and gold theme with smooth animations. Fully responsive.

## Sections

- **Hero** — full-screen headline, animated stats, "Book a Free Trial" CTA
- **Programs** — Kids BJJ, Adult BJJ, Competition Team
- **Coaches** — instructor profiles
- **Schedule** — weekday / weekend class tabs with Kids/Adult tags
- **Testimonials** — athlete and parent reviews
- **Pricing** — three membership tiers
- **Contact** — details and free trial booking form

## Run locally

Open `index.html` in your browser, or serve with any static server:

```bash
npx serve .
```
